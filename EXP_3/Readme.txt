Compile and run "exp3.cpp" for spatial filtering

Keep all the input images in the same folder as "exp3.cpp"

Input:Move trackbar to select input image, filter type, kernel size and orientation.
Output: Filtered image.

Check console output for selected input image, filter type, kernel size and orientation.

key points:
1. For Gaussian and LOG filter kernel size 5*5 is supported only
2. Diagonal orientation i.e. select_orient 2 is supported only for sobel filter.
